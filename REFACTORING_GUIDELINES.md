## Comments

### Prefix to-do comments with `TODO:`
```cs
// TODO: Do this thing on this method

{
  return;
}
```

### Prefix reference URLs with `REF:`
```cs
// REF: https://www.google.com
public void Method()
{
  return;
}
```
